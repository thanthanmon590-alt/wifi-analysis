# 서울시 공공와이파이 설치 현황과 지역별 접근성 분석 대시보드

본 프로젝트는 서울시 공공와이파이 서비스 위치 정보를 활용하여 공공와이파이의 지역별 설치 현황과 설치 유형을 분석한 시각화 대시보드이다.

## 사용 데이터
- 데이터명: 서울시 공공와이파이 서비스 위치 정보
- 출처: 서울열린데이터광장
- 주요 컬럼: 관리번호, 자치구, 와이파이명, 도로명주소, 설치유형, 설치기관, 설치년도
  
## 데이터 전처리 및 클렌징 과정
원본 CSV 파일은 한글 데이터가 포함되어 있어 인코딩 문제가 발생할 수 있으므로 cp949, euc-kr, utf-8-sig 인코딩을 순서대로 시도하여 데이터를 불러왔다.  
컬럼명 앞뒤 공백을 제거하였고, 자치구, 설치유형, 설치년도 컬럼을 기준으로 분석에 필요한 데이터를 정리하였다.  
설치년도는 숫자형으로 변환한 뒤 결측값을 제거하여 연도별 설치 추이 분석에 활용하였다.
## 분석 내용
1. 자치구별 공공와이파이 설치 수
2. 설치유형별 공공와이파이 분포
3. 설치년도별 공공와이파이 설치 추이

## 시각화 결과 설명
자치구별 설치 수를 통해 공공와이파이가 어느 지역에 많이 설치되어 있는지 확인하였다.  
설치유형별 분포를 통해 공공와이파이가 주로 어떤 장소에 설치되어 있는지 분석하였다.  
설치년도별 추이를 통해 공공와이파이 설치가 어느 시기에 증가했는지 확인하였다.

## 사용한 프롬프트
Google AI Studio와 ChatGPT를 활용하여 Streamlit 대시보드 코드 작성 및 오류 수정 도움을 받았다.  
프롬프트 내용은 서울시 공공와이파이 서비스 위치 정보를 활용하여 자치구별 설치 수, 설치유형별 분포, 설치년도별 설치 추이를 시각화하는 Streamlit 대시보드 코드를 요청하는 내용이었다.
